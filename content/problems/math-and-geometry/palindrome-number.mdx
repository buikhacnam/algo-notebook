## Description
Given an integer <code>x</code>, return <code>true</code>* if *<code>x</code>* is a *<span data-keyword="palindrome-integer">*palindrome*</span>*, and *<code>false</code>* otherwise*.

 
Example 1:

<br /><pre className="overflow-x-auto">
Input: x = 121
Output: true
Explanation: 121 reads as 121 from left to right and from right to left.
</pre>

Example 2:

<br /><pre className="overflow-x-auto">
Input: x = -121
Output: false
Explanation: From left to right, it reads -121. From right to left, it becomes 121-. Therefore it is not a palindrome.
</pre>

Example 3:

<br /><pre className="overflow-x-auto">
Input: x = 10
Output: false
Explanation: Reads 01 from right to left. Therefore it is not a palindrome.
</pre>

 
Constraints:


	- <code>-2<sup>31</sup> &lt;= x &lt;= 2<sup>31</sup> - 1</code>


 
Follow up: Could you solve it without converting the integer to a string?

## Solution

### Javascript
```javascript
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function (x) {
    // Creates array from int characters
    // 121 -> [1,2,1]
    let arr = Array.from(String(x), Number);

    // Uses two pointer
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] !== arr[arr.length - 1 - i]) {
            return false;
        }
    }

    return true;
};

/**
 * Reverse Integer Using Modulo
 * Time O(N) | Space O(1)
 * https://leetcode.com/problems/palindrome-number/
 * @param {number} x
 * @return {boolean}
 */
 var isPalindrome = function(x) {
    if (x < 0) return false;

    const inputX = x;
    let revX = 0;

    while (x > 0) {
        revX += x % 10;
        x = Math.floor(x / 10);

        if (x > 0) revX *= 10
    }

    return revX === inputX;
};
```
