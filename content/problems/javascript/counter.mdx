## Description

Given an integer <code>n</code>, return a <code>counter</code> function. This <code>counter</code> function initially returns <code>n</code> and then returns 1 more than the previous value every subsequent time it is called (<code>n</code>, <code>n + 1</code>, <code>n + 2</code>, etc).

Example 1:

<br />
<pre className="overflow-x-auto">
	Input: n = 10 [&quot;call&quot;,&quot;call&quot;,&quot;call&quot;] Output:
	[10,11,12] Explanation: counter() = 10 // The first time counter() is called,
	it returns n. counter() = 11 // Returns 1 more than the previous time.
	counter() = 12 // Returns 1 more than the previous time.
</pre>

Example 2:

<br />
<pre className="overflow-x-auto">
	Input: n = -2
	[&quot;call&quot;,&quot;call&quot;,&quot;call&quot;,&quot;call&quot;,&quot;call&quot;]
	Output: [-2,-1,0,1,2] Explanation: counter() initially returns -2. Then
	increases after each sebsequent call.
</pre>

Constraints:

    - <code>-1000<sup> </sup>&lt;= n &lt;= 1000</code>
    - <code>At most 1000 calls to counter() will be made</code>

## Solution

### Javascript

```javascript
/**
 * @param {number} n
 * @return {Function} counter
 */
var createCounter = function(n) {
    return function() {
        return n++;
    };
};
```
