## Description

Write a function <code>createHelloWorld</code>. It should return a new function that always returns <code>&quot;Hello World&quot;</code>.

Example 1:

<br />

<pre className="overflow-x-auto">
	Input: args = [] Output: &quot;Hello World&quot; Explanation: const f =
	createHelloWorld(); f(); // &quot;Hello World&quot; The function returned by
	createHelloWorld should always return &quot;Hello World&quot;.
</pre>

Example 2:

<br />

<pre className="overflow-x-auto">
	Input: args = [{},null,42] Output: &quot;Hello World&quot; Explanation: const
	f = createHelloWorld(); f({}, null, 42); // &quot;Hello World&quot; Any
	arguments could be passed to the function but it should still always return
	&quot;Hello World&quot;.
</pre>

Constraints:

    - <code>0 &lt;= args.length &lt;= 10</code>
