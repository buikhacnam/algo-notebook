## Description
Given a multi-dimensional array of integers, return a generator object which yields integers in the same order as inorder traversal.

A multi-dimensional array is a recursive data structure that contains both integers and other multi-dimensional arrays.

inorder traversal iterates over each array from left to right, yielding any integers it encounters or applying inorder traversal to any arrays it encounters.

 
Example 1:

<br /><pre className="overflow-x-auto">
Input: arr = [[[6]],[1,3],[]]
Output: [6,1,3]
Explanation:
const generator = inorderTraversal(arr);
generator.next().value; // 6
generator.next().value; // 1
generator.next().value; // 3
generator.next().done; // true
</pre>

Example 2:

<br /><pre className="overflow-x-auto">
Input: arr = []
Output: []
Explanation: There are no integers so the generator doesn&#39;t yield anything.
</pre>

 
Constraints:


	- <code>0 &lt;= arr.flat().length &lt;= 10<sup>5</sup></code>
	- <code>0 &lt;= arr.flat()[i] &lt;= 10<sup>5</sup></code>
	- <code>maxNestingDepth &lt;= 10<sup>5</sup></code>


 
Can you solve this without creating a new flattened version of the array?
