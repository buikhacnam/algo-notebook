## Description

Given an integer array <code>arr</code> and a mapping function <code>fn</code>, return a new array with a transformation applied to each element.

The returned array should be created such that <code>returnedArray[i] = fn(arr[i], i)</code>.

Please solve it without the built-in <code>Array.map</code> method.

Example 1:

<br /><pre className="overflow-x-auto">
Input: arr = [1,2,3], fn = function plusone(n) { return n + 1; }
Output: [2,3,4]
Explanation:
const newArray = map(arr, plusone); // [2,3,4]
The function increases each value in the array by one. 
</pre>

Example 2:

<br /><pre className="overflow-x-auto">
Input: arr = [1,2,3], fn = function plusI(n, i) { return n + i; }
Output: [1,3,5]
Explanation: The function increases each value by the index it resides in.
</pre>

Example 3:

<br /><pre className="overflow-x-auto">
Input: arr = [10,20,30], fn = function constant() { return 42; }
Output: [42,42,42]
Explanation: The function always returns 42.
</pre>

Constraints:

    - <code>0 &lt;= arr.length &lt;= 1000</code>
    - <code><font face="monospace">-10<sup>9</sup> &lt;= arr[i] &lt;= 10<sup>9</sup></font></code>
    - <font face="monospace"><code>fn returns a number</code></font>
