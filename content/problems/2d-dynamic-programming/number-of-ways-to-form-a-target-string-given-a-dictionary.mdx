## Description
You are given a list of strings of the same length <code>words</code> and a string <code>target</code>.

Your task is to form <code>target</code> using the given <code>words</code> under the following rules:


	- <code>target</code> should be formed from left to right.
	- To form the <code>i<sup>th</sup></code> character (0-indexed) of <code>target</code>, you can choose the <code>k<sup>th</sup></code> character of the <code>j<sup>th</sup></code> string in <code>words</code> if <code>target[i] = words[j][k]</code>.
	- Once you use the <code>k<sup>th</sup></code> character of the <code>j<sup>th</sup></code> string of <code>words</code>, you can no longer use the <code>x<sup>th</sup></code> character of any string in <code>words</code> where <code>x &lt;= k</code>. In other words, all characters to the left of or at index <code>k</code> become unusuable for every string.
	- Repeat the process until you form the string <code>target</code>.


Notice that you can use multiple characters from the same string in <code>words</code> provided the conditions above are met.

Return *the number of ways to form <code>target</code> from <code>words</code>*. Since the answer may be too large, return it modulo <code>10<sup>9</sup> + 7</code>.
