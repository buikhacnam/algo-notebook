## Description
You are given a 0-indexed binary string <code>s</code> and two integers <code>minJump</code> and <code>maxJump</code>. In the beginning, you are standing at index <code>0</code>, which is equal to <code>&#39;0&#39;</code>. You can move from index <code>i</code> to index <code>j</code> if the following conditions are fulfilled:


	- <code>i + minJump &lt;= j &lt;= min(i + maxJump, s.length - 1)</code>, and
	- <code>s[j] == &#39;0&#39;</code>.


Return <code>true</code><i> if you can reach index </i><code>s.length - 1</code><i> in </i><code>s</code>*, or *<code>false</code>* otherwise.*

 
Example 1:

<br /><pre className="overflow-x-auto">
Input: s = &quot;<u>0</u>11<u>0</u>1<u>0</u>&quot;, minJump = 2, maxJump = 3
Output: true
Explanation:
In the first step, move from index 0 to index 3. 
In the second step, move from index 3 to index 5.
</pre>

Example 2:

<br /><pre className="overflow-x-auto">
Input: s = &quot;01101110&quot;, minJump = 2, maxJump = 3
Output: false
</pre>

 
Constraints:


	- <code>2 &lt;= s.length &lt;= 10<sup>5</sup></code>
	- <code>s[i]</code> is either <code>&#39;0&#39;</code> or <code>&#39;1&#39;</code>.
	- <code>s[0] == &#39;0&#39;</code>
	- <code>1 &lt;= minJump &lt;= maxJump &lt; s.length</code>
