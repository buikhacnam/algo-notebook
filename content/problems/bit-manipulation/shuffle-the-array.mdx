## Description
Given the array <code>nums</code> consisting of <code>2n</code> elements in the form <code>[x<sub>1</sub>,x<sub>2</sub>,...,x<sub>n</sub>,y<sub>1</sub>,y<sub>2</sub>,...,y<sub>n</sub>]</code>.

*Return the array in the form* <code>[x<sub>1</sub>,y<sub>1</sub>,x<sub>2</sub>,y<sub>2</sub>,...,x<sub>n</sub>,y<sub>n</sub>]</code>.

 
Example 1:

<br /><pre className="overflow-x-auto">
Input: nums = [2,5,1,3,4,7], n = 3
Output: [2,3,5,4,1,7] 
Explanation: Since x<sub>1</sub>=2, x<sub>2</sub>=5, x<sub>3</sub>=1, y<sub>1</sub>=3, y<sub>2</sub>=4, y<sub>3</sub>=7 then the answer is [2,3,5,4,1,7].
</pre>

Example 2:

<br /><pre className="overflow-x-auto">
Input: nums = [1,2,3,4,4,3,2,1], n = 4
Output: [1,4,2,3,3,2,4,1]
</pre>

Example 3:

<br /><pre className="overflow-x-auto">
Input: nums = [1,1,2,2], n = 2
Output: [1,2,1,2]
</pre>

 
Constraints:


	- <code>1 &lt;= n &lt;= 500</code>
	- <code>nums.length == 2n</code>
	- <code>1 &lt;= nums[i] &lt;= 10^3</code>


## Solution

### Javascript
```javascript
/**
 * @param {number[]} nums
 * @param {number} n
 * @return {number[]}
 */
var shuffle = function (nums, n) {
    for (let i = 0; i < n; i++) {
        nums[i] = nums[i] << 10;
        nums[i] = nums[i] | nums[i + n];
    }

    let j = 2 * n - 1;

    for (let i = n - 1; i > -1; i--) {
        let y = nums[i] & (2 ** 10 - 1);
        let x = nums[i] >> 10;
        nums[j] = y;
        nums[j - 1] = x;
        j -= 2;
    }

    return nums;
};
```
