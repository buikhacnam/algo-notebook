## Description

There is a function <code>signFunc(x)</code> that returns:

    - <code>1</code> if <code>x</code> is positive.
    - <code>-1</code> if <code>x</code> is negative.
    - <code>0</code> if <code>x</code> is equal to <code>0</code>.

You are given an integer array <code>nums</code>. Let <code>product</code> be the product of all values in the array <code>nums</code>.

Return <code>signFunc(product)</code>.

Example 1:

<br />
<pre className="overflow-x-auto">
	Input: nums = [-1,-2,-3,-4,3,2,1] Output: 1 Explanation: The product of all
	values in the array is 144, and signFunc(144) = 1
</pre>

Example 2:

<br />
<pre className="overflow-x-auto">
	Input: nums = [1,5,0,2,-3] Output: 0 Explanation: The product of all values in
	the array is 0, and signFunc(0) = 0
</pre>

Example 3:

<br />
<pre className="overflow-x-auto">
	Input: nums = [-1,1,-1,1,-1] Output: -1 Explanation: The product of all values
	in the array is -1, and signFunc(-1) = -1
</pre>

Constraints:

    - <code>1 &lt;= nums.length &lt;= 1000</code>
    - <code>-100 &lt;= nums[i] &lt;= 100</code>

## Solution

### Javascript

```javascript
const arraySign = function (nums) {
	let sign = 1

	for (const num of nums) {
		if (num == 0) return 0
		if (num < 0) sign = -1 * sign
	}

	return sign
}
```

### Python

```python
class Solution:
    def arraySign(self, nums: List[int]) -> int:
        flag = True
        for i in nums:
            if i == 0:
                return 0
            if i < 0:
                flag = not flag

        return 1 if flag else -1

```

class Solution {
    public int arraySign(int[] nums) {
        int sign = 1;
        for (int i : nums) {
            if (i == 0) return 0;
            if (i < 0) sign *= -1;
        }
        return sign;
    }
}
