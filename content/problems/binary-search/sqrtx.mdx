## Description
Given a non-negative integer <code>x</code>, return *the square root of *<code>x</code>* rounded down to the nearest integer*. The returned integer should be non-negative as well.

You must not use any built-in exponent function or operator.


	- For example, do not use <code>pow(x, 0.5)</code> in c++ or <code>x ** 0.5</code> in python.


 
Example 1:

<br /><pre className="overflow-x-auto">
Input: x = 4
Output: 2
Explanation: The square root of 4 is 2, so we return 2.
</pre>

Example 2:

<br /><pre className="overflow-x-auto">
Input: x = 8
Output: 2
Explanation: The square root of 8 is 2.82842..., and since we round it down to the nearest integer, 2 is returned.
</pre>

 
Constraints:


	- <code>0 &lt;= x &lt;= 2<sup>31</sup> - 1</code>



## Solution

### Javascript
```javascript
/**
 * Binary Search 
 * https://leetcode.com/problems/sqrtx/
 * 
 * Time O(log(n)) | Space O(1)
 * @param {number} x
 * @return {number}
 */
var mySqrt = function(x) {
    let left = 1;
    let right = x;
 
    while(left <= right) {
        const mid = (left + right) >> 1;
        if(mid * mid <= x && (mid+1) * (mid+1) > x) return mid;
        if(mid * mid < x) {
            left = mid + 1;
        } else {
            right = mid -1;
        }
    } 
 
    return 0;
 };

```

```python
        l, r = 0, x

            mid = (l + r) // 2
            if mid * mid == x:
                return mid

                l = mid + 1
            else:
                r = mid - 1
        return r
```

```cpp
class Solution {
public:
    int mySqrt(int x) {

        int l = 0;
        int r = x;

        while(l <= r){
            long int m = (l + r) / 2;
            if(m * m == x){
                return m;
            }
            else if(m * m > x){
                r = m - 1;
            }
            else{
                l = m + 1;
            }
        }
        return r;
    }
};
```

### Java
```java
class Solution {
    public int mySqrt(int x) {
        
        // Linear search way -> loop through for all nums till x
        // then see if their square <= x

        // Binary Search way -> we can optimise our approach by observing
        // that nums till x are sorted

        int left = 0;
        int right = x;
        int mid = 0;
        int probableAns = 0;

        while(left <= right){
            mid = left + (right-left)/2;
            if((long)mid*mid <= (long)x){
                probableAns = mid;
                // let's see if we can find a bigger num
                left = mid+1;
            }
            else if((long)mid*mid > (long)x){
                right = mid-1;
            }
        }

        return probableAns;
    }
}
```
