## Description
Given an integer <code>n</code>, return *all the structurally unique BST&#39;s (binary search trees), which has exactly *<code>n</code>* nodes of unique values from* <code>1</code> *to* <code>n</code>. Return the answer in any order.

 
Example 1:

</pre>
